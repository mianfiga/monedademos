monedademos
===========

The core of the alternative currency (social coin) that runs at http://monedademos.es

You want to start with your currency? Please let us know at contacto@monedademos.es. Anyway you will have to:

路 Put this all in a server (you will also need Yii Framework Version 1.1.12 vailable here: https://github.com/yiisoft/yii/releases/tag/1.1.12 in a sibling "Yii" folder).  
路 Set up a database with the sql at /protected/data/rbu.sql  
路 Configure (at least) your database at /protected/configure/ main.php, console.php (use the corresponding *_example.php file as a guide).
路 Make sure both folders assets/ and protected/runtime are writable by web server.

There are many thing to do. Would you join us?
